----------------------
ランキングエリア追加
----------------------
■対象ファイル
・addRanking.js
・addRanking.css

■実装エリア
ファッション通販「MAGASEEK」のアプリトップページ

■制作日
2023年8月9日

■制作背景
マガシークアプリ改善プロジェクトにて実装。
元々アプリトップページのコンテンツ量が少ないという課題から、受注インパクトが高いランキングエリアを追加する案を自ら提出し、
デザイン案から実装までを一貫して対応。
カテゴリ別に商品を表示させるとページ高さがででしまい、UI面で懸念があるため、
上部ナビでカテゴリを選択できるようにし、選択されているカテゴリに対してAPIを読み込む処理を実施した。

■効果
エリア追加後のランキング経由受注が月300万～400万円と売り上げに貢献

予約アイテムエリア追加
----------------------
■対象ファイル
・addReseve.js
・addReseve.css

■実装エリア
ファッション通販「MAGASEEK」サイトのトップページ
https://www.magaseek.com/top/index/tp_1

■制作日
2023年9月19日

■チーム
・営業：1名
・ディレクター：1名
・コーダー：1名

■制作背景
営業部門から、プロパー予約アイテムの受注強化の依頼を受け、トップページに予約エリアを追加することとなりました。
以下の要望を元に実装を行いました。

■要望
①一押しショップをランダムで掲出したい
②特集バナーをランダムで掲出したい
③リアルタイムで新着アイテムを掲出したい
④レディース・メンズで商材をだし分けたい
⑤ショップの更新が発生するため、更新工数をなるべく抑えたい

■効果
予約プロパーアイテムの受注が増加
